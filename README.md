# Football Analytics

## Εκτέλεση

**1. Ενεργοποίηση virtual environment**
```
venv\Scripts\activate
```

**2. Εγκατάσταση dependencies**
```
pip install -r requirements.txt
```

**3. Εκτέλεση**
```
python main.py
```

## Αποτελέσματα

- Πάτα `q` για να σταματήσεις νωρίτερα.
- Στατιστικά κατοχής εκτυπώνονται στο terminal στο τέλος.
- Τα heatmaps αποθηκεύονται στο `data/` ως `heatmap_team_0.png` και `heatmap_team_1.png`.
